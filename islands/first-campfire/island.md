# First Campfire

## Mission

Create a new Godot project, make a simple scene, and run it.

## Why This Matters

Before you build games, you need to know how to start a project, place something in the world, and see it running.

## You Will Learn

- How to create a Godot project.
- What a scene is.
- How to run your project.

## Requirements

- Create a new Godot project.
- Create one scene.
- Add one visible object to the scene.
- Run the scene without errors.

## Build Steps

1. Open Godot and create a new project.
2. Create a new scene.
3. Add one visible node, like a sprite, mesh, label, or colored shape.
4. Save the scene.
5. Run the project and confirm your object appears.

## Captain's Log

Write down:

- What did you notice in the Godot editor?

## Stretch Goals

- Change the object's color, size, or position.
- Add a second object.
- Rename your nodes so the scene is easy to understand.
