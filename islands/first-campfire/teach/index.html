First Campfire Teaching Notes

Goal: understand the smallest useful Godot loop: create a scene, save it, run it, and observe what happened.

Concept

A Godot project is built from scenes. A scene is a reusable group of nodes. For this island, one scene with one visible node is enough.

Tiny Example

Create a scene with a root node, add something visible like a Label, save the scene, then run it.

Quick Check

If nothing appears when you run the project, check that the scene is saved and selected as the main scene.
