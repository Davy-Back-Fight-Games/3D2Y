# Islands

Islands are small Godot mini-projects. Each one focuses on a practical skill and gives you something finished to show.

Pick any island that matches your current level. If an island lists `requires`, visit those islands first.

Some islands also include a `teach/` folder with short HTML lessons or references for extra support.

## The Loop

1. Read the mission.
2. Build the smallest working version.
3. Check the requirements.
4. Write your Captain's Log.
5. Share what you made with the crew.
