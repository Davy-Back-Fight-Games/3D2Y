# 3D2Y Resources

Use these sources when writing islands and `teach/*.html` support files. Prefer official, versioned Godot documentation when explaining engine behavior.

## Knowledge

- [Godot 4.7 Documentation](https://docs.godotengine.org/en/4.7/)
  Use for: version-specific Godot explanations, tutorials, and API references.

- [Godot 4.7: Step by Step](https://docs.godotengine.org/en/4.7/getting_started/step_by_step/index.html)
  Use for: beginner explanations of nodes, scenes, scripts, input, and signals.

- [Godot 4.7: Nodes and Scenes](https://docs.godotengine.org/en/4.7/getting_started/step_by_step/nodes_and_scenes.html)
  Use for: explaining the core scene/node model behind most islands.

- [Godot 4.7: Your First 2D Game](https://docs.godotengine.org/en/4.7/getting_started/first_2d_game/index.html)
  Use for: beginner-friendly examples of project structure, player movement, enemies, UI, and scoring.

- [Godot Demo Projects](https://github.com/godotengine/godot-demo-projects)
  Use for: complete working examples when an island needs a reference implementation or inspiration.

## Wisdom: Communities

- [Godot Forum](https://forum.godotengine.org/)
  Use for: troubleshooting, project feedback, and engine-specific questions.

- [Godot Contributors Chat](https://chat.godotengine.org/)
  Use for: documentation and contributor discussions.
