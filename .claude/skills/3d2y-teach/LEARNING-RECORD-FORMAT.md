# Learning Record Format

Learning records live in `learning-records/` and capture durable teaching decisions or learner insights for 3D2Y.

Use sequential filenames:

```txt
learning-records/0001-short-slug.md
learning-records/0002-short-slug.md
```

## Template

```md
# Short title

One to three sentences describing what was learned or decided, why it matters, and how it should affect future islands or teaching files.
```

## When To Write One

Write a learning record when:

- A common learner misconception appears.
- An island structure decision should guide future islands.
- The project audience or difficulty model changes.
- A teaching approach proves especially useful or harmful.

Do not write learning records for routine activity logs.
