# Mission Format

3D2Y's mission is project-level, not private to one learner.

If a mission file is needed, create `MISSION.md` at the repository root using this shape.

## Template

```md
# Mission: 3D2Y

## Why

3D2Y helps learners build Godot skills through small finished mini-projects, framed as islands in a timeskip training journey.

## Success Looks Like

- Learners can pick an island and finish a tiny playable project.
- Each island teaches a practical Godot skill without becoming a long course chapter.
- The community can share builds, feedback, and new island ideas.

## Constraints

- Keep the repo GitHub-native first.
- Keep island metadata minimal and platform-ready.
- Prefer public reusable learning material over private notes.

## Out Of Scope

- Building a custom platform before the learning loop is useful.
- Writing exhaustive tutorials for every Godot feature.
```
