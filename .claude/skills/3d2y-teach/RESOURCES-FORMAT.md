# RESOURCES.md Format

`RESOURCES.md` lives at the repository root. It is the curated source list for 3D2Y teaching material.

Use it to keep island lessons grounded in trusted knowledge instead of guesses. Prefer official, versioned Godot documentation that matches an island's `godot_version`.

## Template

```md
# 3D2Y Resources

## Knowledge

- [Godot Docs: Your First 2D Game](https://docs.godotengine.org/)
  Use for: beginner-friendly scene, node, input, and game-loop explanations.

## Wisdom: Communities

- [Godot Forum](https://forum.godotengine.org/)
  Use for: real-world troubleshooting, design feedback, and engine-specific questions.

## Gaps

- Need a trusted source for beginner-friendly 3D movement in Godot 4.
```

## Rules

- Prefer official docs, recognized experts, maintained examples, and well-moderated communities.
- Annotate every link with what it is useful for.
- Group resources by `Knowledge` and `Wisdom: Communities`.
- Add `Gaps` when the repo needs a source but does not have one yet.
- Remove weak or outdated resources instead of collecting links forever.
