# 3D2Y

A Godot learning journey inspired by the timeskip: small challenges, steady practice, and a crew learning together.

## What This Is

3D2Y is a community-driven challenge list for learning game development by doing.

Each challenge is a mini-project focused on one practical skill, from moving a character to finishing a tiny playable game.

## The Loop

1. Pick a challenge.
2. Build the smallest working version.
3. Share what you made.
4. Learn from feedback.
5. Move to the next island.

## Goals

- Learn Godot through finished mini-projects.
- Keep challenges small, focused, and replayable.
- Build teaching notes alongside each project.
- Grow a friendly community of learners and makers.

## Status

This repo is just getting started. The first challenge list and contribution guide are coming next.
